# Source Accum3 Pulse

Implement a 3-bit modulo accumulator pulse generator. The internal count starts at 7, increments on each rising edge, and OUT is high only when count is 0.

The module name and port list must match `accum3_pulse.va`. Keep the implementation deterministic and voltage-domain only. The historical source normalized for this task is `huangsy/ACCUM_3_bit.va`.
