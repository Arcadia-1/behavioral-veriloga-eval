Implement an 8-input thermometer-popcount encoder. Count high thermometer inputs and drive the count as a 4-bit voltage-coded binary value.

The module name and port list must match `therm8_to_bin4_count.va`. Keep the implementation deterministic and voltage-domain only. The historical source normalized for this task is `liudongyang/tb_therm8_to_bin4.va`.
