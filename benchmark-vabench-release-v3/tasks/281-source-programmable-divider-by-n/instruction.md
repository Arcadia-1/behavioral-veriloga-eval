# Source Programmable Divider By N

Implement a programmable clock divider. The output is high only when the internal edge counter is zero; with DIVCTRL=3 it asserts every third rising edge.

The module name and port list must match `programmable_divider_by_n.va`. Keep the implementation deterministic and voltage-domain only. The historical source normalized for this task is `zhangm/divider.va`.
