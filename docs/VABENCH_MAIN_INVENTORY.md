## Purpose

future work does not confuse a missing source directory with a validated
benchmark asset.

## Promotion Gate For New Tasks

A task can enter the paper-facing benchmark only after:

1. `prompt.md`, `meta.json`, `checks.yaml`, and `gold/` are complete.
2. Metadata passes schema validation and the task ID matches its directory.
3. Gold EVAS validation passes.
4. Gold Spectre validation passes when the task is paper-facing.
5. The task has a clear coverage purpose and does not duplicate an existing
   case without adding a new circuit or simulator semantic.
